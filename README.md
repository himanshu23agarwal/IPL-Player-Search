# IPL Player Search App

A web-based application to search IPL players based on different criteria such as name, country, and bid amount.

#Features

- Search players by name (case-insensitive)
- Search players by country
- Search players by minimum bid amount
- Dynamic input handling in frontend
- MongoDB database integration

# Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- HTML
- CSS
